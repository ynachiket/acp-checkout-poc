## üéì Key Concepts Explained

### Agentic Commerce Protocol (ACP)
OpenAI's open standard for AI agents to complete purchases.

**Learn more:** 
- ARCHITECTURE.md ‚Üí Appendix
- [OpenAI Docs](https://developers.openai.com/commerce)

### Model Context Protocol (MCP)
Standard for AI agents to discover and invoke tools.

**Learn more:**
- DECISION_LOG.md ‚Üí Decision 4
- [MCP Spec](https://modelcontextprotocol.io)

### Gateway Pattern
Architectural pattern that translates between protocols and services.

**Learn more:**
- ARCHITECTURE_COMPARISON.md ‚Üí Approach 3
- DECISION_LOG.md ‚Üí Decision 3

### Protocol-Agnostic Core
Business logic independent of external protocols.

**Learn more:**
- ARCHITECTURE.md ‚Üí Design Principles
- EXECUTIVE_SUMMARY.md ‚Üí Key Architectural Principle
